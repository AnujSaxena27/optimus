# Quick Start Guide\n\n## 5-Minute Setup\n\n### 1. Prerequisites\n\n```bash\n# Check Node.js version (need 20+)\nnode --version\n\n# Check npm\nnpm --version\n```\n\n### 2. Clone/Setup\n\n```bash\n# Navigate to project\ncd sepmpro\n\n# Install dependencies\nnpm install\n```\n\n### 3. Configure Environment\n\n```bash\n# Copy example configuration\ncp .env.example .env.local\n\n# Edit with your Groq API key and Supabase credentials\n# nano .env.local\n```\n\n**Required Variables**:\n```\nGROQ_API_KEY=gsk_xxxxxxxxxxxxxx\nNEXT_PUBLIC_SUPABASE_URL=https://xxxxx.supabase.co\nNEXT_PUBLIC_SUPABASE_ANON_KEY=xxxxx\n```\n\n### 4. Start Development Server\n\n```bash\nnpm run dev\n```\n\nVisit: http://localhost:3000\n\n### 5. Test API\n\n```bash\n# Get available models\ncurl http://localhost:3000/api/models\n\n# Test chat (requires JWT token)\ncurl -X POST http://localhost:3000/api/chat \\\n  -H \"Content-Type: application/json\" \\\n  -H \"Authorization: Bearer YOUR_JWT_TOKEN\" \\\n  -d '{\n    \"message\": \"Hello!\",\n    \"model\": \"mixtral-8x7b-32768\"\n  }'\n```\n\n---\n\n## Docker Quick Start\n\n```bash\n# Build image\ndocker build -t inference-optimizer .\n\n# Run container\ndocker run -p 3000:3000 \\\n  -e GROQ_API_KEY=gsk_xxxxx \\\n  -e NEXT_PUBLIC_SUPABASE_URL=https://xxxxx.supabase.co \\\n  -e NEXT_PUBLIC_SUPABASE_ANON_KEY=xxxxx \\\n  inference-optimizer\n\n# Or use Docker Compose\ndocker-compose up\n```\n\n---\n\n## Project Structure\n\n```\n├── src/\n│   ├── app/            # Reuse responses longer\nRATE_LIMIT_MAX_REQUESTS=50  # Reduce usage\n```\n\nSee [CONFIGURATION.md](./CONFIGURATION.md) for complete options.\n\n---\n\n## Development Workflow\n\n### 1. Wait and retry.\n\n```bash\n# Check your rate limit settings\ngrep RATE_LIMIT .env.local\n```\n\n### \"Cache not working\"\n\nCheck cache settings:\n\n```bash\n# Increase TTL\nCACHE_TTL_SECONDS=7200  # 2 hours\n\n# Or disable/enable in code\ncacheService.get(model, message, temp) // returns cached result\n```\n\n---\n\n## Next Steps\n\n1. **Read Architecture** → [ARCHITECTURE.md](./ARCHITECTURE.md)\n2. **Configure Settings** → [CONFIGURATION.md](./CONFIGURATION.md)\n3. **Explore API** → [API_DOCS.md](./API_DOCS.md)\n4. **Deploy** → [DEPLOYMENT.md](./DEPLOYMENT.md)\n\n---\n\n## Resources\n\n- **Groq API**: https://console.groq.com\n- **Supabase**: https://supabase.com\n- **Next.js Docs**: https://nextjs.org/docs\n- **Docker Docs**: https://docs.docker.com\n\n---\n\n## Support\n\nFor issues or questions:\n1.
